# 自定义主菜单背景功能

## 功能说明

这个功能允许你为Gemini客户端的主菜单设置自定义背景图片，支持视差效果（鼠标移动时背景会轻微移动）。

## 使用方法

### 1. 准备背景图片

- 支持的格式：PNG、JPG/JPEG
- 推荐尺寸：与你的屏幕分辨率相同或更大（例如：1920x1080或更高）
- 图片会自动缩放并居中显示

### 2. 放置背景图片

将你的背景图片重命名为以下任一名称：
- `custom_bg.png`
- `custom_bg.jpg`
- `custom_bg.jpeg`

然后将图片放到以下目录：
```
<游戏目录>/config/gemini/
```

对于开发环境，路径是：
```
run/client/config/gemini/
```

### 3. 启用背景

启动游戏后，在主菜单左上角会看到一个 **"BG"** 按钮：
- **灰色按钮** = 背景关闭（显示默认网格背景）
- **蓝色按钮** = 背景开启（显示自定义背景）

点击按钮即可切换背景开关状态。

### 4. 视差效果

启用自定义背景后，移动鼠标可以看到背景会轻微移动，产生视差效果。

## 配置文件

配置保存在：
```
<游戏目录>/config/gemini/background.json
```

内容示例：
```json
{
  "customBackgroundEnabled": false
}
```

你可以手动编辑这个文件来开启/关闭背景（需要重启游戏）。

## 故障排除

### 背景不显示

1. 确认图片文件名正确（`custom_bg.png`、`custom_bg.jpg` 或 `custom_bg.jpeg`）
2. 确认图片放在正确的目录
3. 确认点击了BG按钮启用了背景
4. 如果BG按钮上有 **红色感叹号(!)**, 表示找不到背景图片文件

### 图片显示异常

- 如果图片太小，可能会被放大导致模糊
- 如果图片尺寸与屏幕比例不同，边缘可能会被裁剪
- 建议使用与屏幕分辨率相同或更大的图片

## 技术细节

- 背景图片会被缩放至屏幕的110%以覆盖视差移动范围
- 视差偏移最大为40像素（根据鼠标位置）
- 背景渲染在网格背景之前，所以网格会叠加在自定义背景之上
- 图片加载失败会被记录，不会导致游戏崩溃

## 代码实现

主要涉及两个文件：
1. `BackgroundConfig.java` - 配置管理
2. `MainMenuScreen.java` - 背景渲染和UI

功能完全独立，不影响其他游戏功能。
